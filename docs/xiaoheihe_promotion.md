## 这个叫 Pulse 的东西是什么？

一句话：**它是个住在你电脑里的 AI 助手，开源、免费、不用开会员，而且你的聊天记录全留在本地，不出门。**

它跟那些要你充 VIP、动不动就"服务器繁忙"的在线助手不一样——Pulse 默认就跑在你自己的机器上，用的是 Ollama 这类本地大模型。简单说就是：**你自己当家做主，不依赖任何厂家的云。**

## 怎么玩起来（诚实版）

说句实话：Pulse 目前是个命令行工具，不是点几下就能用的 App。你需要：

1. 电脑装个 [Python](https://www.python.org/)（大部分开发向的玩家机器上都有）
2. 装个本地模型运行环境 [Ollama](https://ollama.com/)
3. 敲几行命令把它跑起来

```bash
# 装好之后大概是这样
pulse init --yes
pulse chat "帮我写一份原神枫丹版本的探索路线"
```

不懂命令也没关系，项目里带了"一键配置"向导，基本是回车到底。对纯小白来说有点门槛，但对愿意折腾的玩家来说，这反而是乐趣——**像是自己 DIY 了个专属 AI 队友。**

> 如果实在不想碰命令行，也可以切成"演示模式"先感受一下它脑子里装了啥。
