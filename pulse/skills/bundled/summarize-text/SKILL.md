# Summarize Text

Distill any long input into a concise, structured brief.

## Steps
1. Identify the core thesis / goal of the text.
2. Extract 3–5 key points with supporting detail.
3. Note any open questions or decisions required.
4. Output a short summary (≤150 words) with a bullet list.

## Notes
- Preserve named entities, numbers, and action items exactly.
- If the text is ambiguous, state assumptions explicitly.
